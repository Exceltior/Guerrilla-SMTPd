Guerrilla-SMTPd by flashmob

SMTP server made for receiving large volume of mail.

Guerrilla SMTPd is an SMTP mail server, made with the goal of receiving large volumes of email. So far, it has been interesting experiment to see what PHP can do. It's currently used on www.guerrillamail.com

Demo

Click the 'About' tab to view the stats - it shows show how many emails go in every hour.

Dependencies

Needs PHP with the socket library enabled. Also needs Mb_string and Mailparse extensions
